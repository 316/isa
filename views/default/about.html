Sobre Isa

Isa es un sistema de gestion para multiples eventos, que nacio del cansancio de adaptar una aplicacion especifica para estas actividades n veces. Si tienes un LUG, grupo de hackers, nerds, o solo te gusta amontonar gente para hacer cosas, sientete libre de usarlo puedes hacer muchas reuniones con el mismo sitio!, y la gente que siempre va se registra y los nuevos tambien!

Fue inspirado por Sofi, software para gestion de eventos escrito en django, pero con algunos features menos, que aun se encuentra en desarrollo

El nombre esta inspirado en mi hija, que se llama Isabella.

Habemos participado dos personas en el desarrollo de esta sencilla aplicacion:
